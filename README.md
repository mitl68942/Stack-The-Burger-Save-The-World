# 🍔《叠堡之怒：雷德王覆灭记》游戏简介

## 🌍 背景
雷德王正以毁灭姿态突破大气层，人类的所有防线都已失效。  
最后一道希望，是一只名叫**奶龙**的小家伙，以及它手中那座不断生长的**汉堡塔**。  
只有搭起足够高、足够稳固的汉堡，积蓄**止战之力**，奶龙才能觉醒力量，击退雷德王。

## 🎮 核心玩法
玩家控制奶龙左右移动（鼠标 / A D / ← → / 手指拖拽），接住从天空随机掉落的食材，一層一層堆叠汉堡。

### 🥩 可用食材（共5种）
- 🥩 **牛肉肉饼** —— 基础分 40，止战值的核心来源
- 🍍 **菠萝** —— 基础分 20
- 🍅 **番茄片** —— 基础分 15
- 🥬 **生菜** —— 基础分 10
- 🍞 **汉堡顶** —— 特殊食材，**只能且必须在第36层（最后一层）接住**，提前接则立即失败

### 🎯 胜利条件（必须同时满足）
1. 汉堡总层数达到 **36 层**  
2. 总止战值（所有食材得分总和）≥ **1000**  
3. 第36层必须是 **汉堡顶**  

### 💀 失败条件（满足任意一条即游戏结束）
- 任何食材（包括汉堡顶）**掉落地面**未被接住  
- 汉堡**动荡值 ≥ 3**（汉堡倒塌）  
- 在达到36层前**提前接住汉堡顶**  
- 达到36层但止战值 **＜ 1000**  

## ⚙️ 核心机制详解

### 1️⃣ 完美系数（0~100）
每次接住食材时，根据**与上一层中心的偏移程度**计算：
- 越靠近中心 → 系数越高  
- 系数直接影响 **得分倍率**（1倍～3倍基础分）  
- 同时影响 **不稳值** 的变化

| 完美系数 | 评级 | 不稳值变化 |
| --- | --- | --- |
| 85～100 | Perfect | +0 |
| 70～84 | Good | +0 |
| 40～69 | Shaky | +1 |
| 0～39 | Danger | +3 |

> 💡 例：一块牛肉肉饼，Perfect时可获得 40×3 = 120 止战值；Danger时仅得 40×1 = 40。

### 2️⃣ 动荡值系统
- 初始为 **0**，上限为 **3**  
- 每得到 Shaky (+1) 或 Danger (+3) 都会累加  
- **一旦达到 3 → 汉堡立即倒塌，游戏失败**  
- 连续 **3 次** 获得 Good 或 Perfect → 触发 **稳定连击**，**不稳值直接清零**，同时播放金光特效。

### 3️⃣ 倒计时压力
屏幕顶部显示 **雷德王降临地球倒计时**。倒计时与层数无关，但会给玩家心理压力。  
你必须在倒计时结束前（通常倒计时足够堆36层）完成目标，否则雷德王提前落地？——实际机制中，倒计时结束若还未胜利也会失败（可补充说明：倒计时归零时若未胜利 → 雷德王毁灭地球）。

*（根据你的设定，如果倒计时仅作为氛围提示，可不设为硬性失败；但为了“具体”，建议明确：倒计时归零时若未达成胜利条件，同样失败。）*

## 📊 界面信息（实时显示）
- 当前层数 / 目标36层  
- 当前止战值 / 目标1000  
- 当前完美系数（带评级文字）  
- 动荡值（不稳值）：⚡⚡⚡ 显示  
- 雷德王降临倒计时（秒）

## 💡 具体策略提示
- **牛肉肉饼**是冲分关键，尽量接准拿到 Perfect（120分）  
- 连续两个 Danger（+3+3）会直接崩盘，宁可接偏一点也要避免 Danger 区间  
- 利用 **稳定连击** 清空不稳值：连续3次 Good 或 Perfect 即可“洗白”  
- 第35层时，一定要确保下一层（第36层）是汉堡顶，如果汉堡顶提前出现而层数不足35，可以故意不接（让它掉落？注意：食材掉落也会失败），所以——**必须接住所有非顶食材，只能在第36层接顶**。因此游戏设计上，汉堡顶只会在第36层附近掉落一次。

## 🏆 胜利演出
达成36层 + 止战值1000 + 顶层封顶 →  
🌈 汉堡塔发出耀眼光芒 → 奶龙觉醒，止战之力爆发 → 💥 一道光柱击穿大气层，雷德王被击退 → 🎉 地球和平恢复，显示通关画面。

**一句话总结（具体版）**  
> 控制奶龙精准接住牛肉饼、菠萝、番茄片、生菜，每层对准中心，避免不稳值达到3，在第36层用汉堡顶完美封顶，且总分超过1000，即可觉醒击败雷德王。
